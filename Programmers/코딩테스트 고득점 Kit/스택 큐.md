### [Level 1 같은 숫자는 싫어](https://school.programmers.co.kr/learn/courses/30/lessons/12906)

```text
앞 뒤 숫자 비교
```

```python
def solution(arr):
    answer = []
    tmp = arr[0]
    answer.append(tmp)
    for i in range(1, len(arr)):
        if tmp != arr[i]:
            answer.append(arr[i])
        tmp = arr[i]
    return answer
```

### [Level 2 올바른 괄호](https://school.programmers.co.kr/learn/courses/30/lessons/12909)

```text
백준 괄호 문제와 똑같은 문제
```

```python
def solution(s):
    cnt = 0

    for i in range(len(s)):
        if s[i] == '(':
            cnt += 1
        if s[i] == ')':
            cnt -= 1

        if cnt < 0:
            return False
    if cnt == 0:
        return True
    else:
        return False
```
